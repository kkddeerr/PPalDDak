한성대학교 컴퓨터공학부 W2M3팀의 졸업프로젝트 작품 입니다.

아두이노를 이용한 IOT 스마트 낚시대 팔딱팔딱 프로젝트의 안드로이드 클라이언트입니다.

블루투스를 통해 IOT 스마트 낚시대의 데이터를 수신하며

수신된 데이터로 알고리즘을 수행합니다.

서버와 HTML 통신하여 저장된 사용자, 물고기 데이터를 송수신합니다.

사용 라이브러리 <br>
